一个草原上有500只老虎 一只羊
虎吃羊？
可以吃草生活
虎吃羊， 一整只 ， 吃完后老虎会变成羊

1. 不吃   因为吃了后会变成羊，
   当只有一只虎的时候，肯定吃羊
2. 吃，  因为羊肉好吃
   当有俩只老虎，一只羊时，吃不了
3. 不吃， 因为羊羊那么可爱，做个朋友吧
   当有三只老虎，一只羊时，吃 （因为吃了就变成上面俩只老虎的情况）
   当有四只老虎，一只羊的时候，不吃。因为变成三只老虎一定会被吃
   当有五只老虎，吃，因为四只老虎不吃

   大事情难解决，分成小事情来解决
   大事化小，递归 
   fn = !(fn-1),n是老虎
   边界  n = 1 吃 ,n = 2 不吃    
   500只老虎，不吃


   动态规划(Dynamic-programing)是常考算法，还可以解决上楼梯问题 
   动态规划有三个术语：
   一定有一个 最优子结构   500->400->.......2-> 1
   一定有一个 边界  n = 1 ,吃  n = 2 ,不吃
    有       状态转移公式  例如：虎变羊  fn = !(fn-1)


    楼梯台阶有12级 一步只能走一阶或走俩阶 ，  请问有多少种走法？
    最优子结构 f(12)
    f(12)  f(11)  f(10)
    边界 1个台阶的时候，有一种走法，2个台阶的时候，有俩种走法 
    状态转移公式 f(12) = f(11) + f(12)
